# 🗳️ Blockchain Voting System - Final Year Project

A decentralized voting system built on Ethereum blockchain for secure, transparent, and tamper-proof elections.

## 🏗️ Tech Stack
- **Frontend:** React.js
- **Blockchain:** Ethereum, Solidity, Truffle
- **Tools:** Ganache, MetaMask

## 📁 Project Structure
- `/contracts` - Smart contracts
- `/voting-ui` - React frontend application
- `/migrations` - Deployment scripts

## 🚀 Getting Started
1. Clone repository
2. Run `npm install` in both folders
3. Start Ganache: `npx ganache-cli --port 8545`
4. Deploy contracts: `truffle migrate --reset`
5. Start React app: `npm start`
